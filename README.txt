# ATM Locator with Spring Boot and Apache Camel

This example shows how to work with a simple Apache Camel application using Spring Boot.

## Camel routes

The Camel route is located in the `ATMRoute` class. In this class the route
starts from a ,

 that is routed to a stream endpoint which writes to standard output.

## How to run
In case of login use admin/admin 

You can run this example using

mvn spring-boot:run
